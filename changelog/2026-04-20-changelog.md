# Release Notes (2026-04-20)

This release focuses on enhancing template management with workspace-based imports and improving the user interface with a new SVG favicon and expanded agent details.

## 🚀 Features
* **Templates: Workspace Import Support:** Introduced the ability to import templates directly from a path within a grove's workspace (defaulting to `/.scion/templates`). This provides a local-first alternative to URL-based imports and includes a new UI toggle in the Grove settings page.
* **UI/UX: SVG Vector Favicon:** Replaced the inconsistent emoji-based favicon with a custom SVG seedling graphic, ensuring a crisp and unified brand appearance across all platforms and browsers.
* **Agent Management: Broker Column in Table View:** Added a "Broker" column to the agent list table in the Grove detail view. This makes it easier to identify and access the runtime broker for each agent directly from the table view.
