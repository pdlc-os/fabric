# Release Notes (2026-04-21)

This update focuses on improving visibility into project configurations and upgrading the core agent toolset.

## 🚀 Features
* **Image Build:** Upgraded Git to version 2.54.0 and added `kubectl` to the `core-base` image, enhancing the toolset available in agent environments.

## 🐛 Fixes
* **Workspace Visibility:** Removed restrictions that filtered the `.scion` directory from the workspace file browser and downloads. Users can now inspect and download templates and settings directly through the web interface.
